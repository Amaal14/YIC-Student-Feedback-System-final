YIC Student Feedback System
Project Description

The YIC Student Feedback System is a web-based application that allows students to submit feedback about campus services and academic experience. Admin users can manage, review, edit, and delete feedback submissions.

Technologies Used

CSS3
JavaScript
PHP
MySQL
PDO
Laragon
phpMyAdmin

User Registration
User Login and Logout

Submit Feedback
View Feedback
Admin Dashboard
Edit Feedback
Delete Feedback
Role-Based Access Control
Input Validation
XSS Prevention
Responsive Design
Setup Instructions
Download or clone the project.
Move the project folder into the Laragon www directory.
Start Apache and MySQL using Laragon.
Open phpMyAdmin.
Create a database named:

yic_feedback_system

Import the file:

yic_feedback_system.sql

Open the project in the browser:

http://localhost/YIC-Student-Feedback-System-final/
